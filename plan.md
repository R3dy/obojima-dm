# Plan: Obojima DM Reference Web App — "The Curious World"

## Objective
Build a single-page web app containing all DM-relevant information and artwork from "The Curious World" adventure in *Obojima: Tales From The Tall Grass*.

## Stage 1 — Content Extraction
- Read the PDF to locate "The Curious World" adventure section.
- Extract all DM-relevant text: adventure overview, locations, NPCs, encounters, maps, items, monsters, lore.
- Identify and extract all original artwork from the book.

## Stage 2 — Asset Extraction
- Use PDF processing and image extraction tools to pull artwork from the PDF.
- Generate any missing images that are needed for the web app experience.
- Organize assets into a clean folder structure.

## Stage 3 — Web App Design & Build
- Skill: `vibecoding-webapp-swarm`
- Design an immersive, dark-themed SPA styled like a magical tome/grimoire.
- Sections: Cover, Adventure Overview, Locations, NPCs, Encounters, Monsters, Items, Lore, Gallery.
- Incorporate all extracted artwork as full assets.
- Deploy the final app.

## Stage 4 — Deploy
- Deploy as a static site.
